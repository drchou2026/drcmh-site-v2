> **摘要：** 血精症 hematospermia，是指精液中出現血液，顏色可能是鮮紅、粉紅、暗紅，也可能像咖啡色或鐵鏽色。它看起來很嚇人，但多數偶發血精和發炎、感染、摩擦、近期檢查處置或小血管破裂有關。真正重要的是：血精是第一次還是反覆？有沒有血尿、疼痛、發燒、解尿不適、PSA 異常或泌尿生殖道感染風險？
> 本文由泌尿科專科醫師周孟翰，用診間故事帶你認識血精症的常見原因、警訊、檢查與治療。

## 診間故事：醫師，精液怎麼變成咖啡色？

「醫師，我昨天嚇到整晚睡不著。」

「怎麼了？」

「我射精的時候，精液顏色怪怪的。不是平常的顏色，有點紅，又有點像咖啡色。我嚇到，這樣正常嗎？」

這是泌尿科門診很常見的開場。

病人通常一坐下來，表情就很緊繃。有些人還會拿手機照片給我看，精液呈現粉紅色、暗紅色，或咖啡色。  
另一半也常一起擔心：「為甚麼會這樣？這會不會很嚴重？」

我通常會先說：

「看到精液有血，緊張是很正常的。很多血精是暫時性的，可能和發炎、感染、摩擦或小血管破裂有關。可是，我們還是要把該問的、該檢查的做完，確認有沒有需要治療的原因。」

這篇文章，就是要把這件事講清楚。

## 血精症是什麼？

血精症 hematospermia，也有人寫作 hemospermia，意思是精液裡出現血液。

精液不是只有精子，它還包含攝護腺、儲精囊、輸精管、尿道周邊腺體等分泌物。只要這一路上的任何一段發炎、受傷、血管破裂或受到刺激，都可能讓血液混進精液裡。

血精可能呈現：

* 鮮紅色或粉紅色
* 暗紅色
* 咖啡色、鐵鏽色
* 精液中有血絲或血塊感

顏色不一定代表嚴重程度。鮮紅色常讓人很恐慌，但咖啡色也可能只是比較久的血液氧化後呈現的顏色。

## 多數血精症是良性的，但不能完全不理

血精症最讓人緊張的地方，是它出現得很突然，而且顏色很有衝擊感。

先講讓人安心的一面：  
**如果只是偶發一次、沒有血尿、沒有疼痛、沒有發燒、沒有其他泌尿道症狀時，多數血精症是良性的。**

常見情況包括：

* 攝護腺、精囊或尿道發炎
* 近期性行為或自慰摩擦較強
* 禁慾一段時間後射精
* 近期泌尿道檢查、導尿、膀胱鏡或攝護腺切片
* 小血管暫時破裂

但另一面也要說清楚：  
**如果反覆發作、持續沒有改善，或合併其他警訊，就不能只說「再觀察」。**

泌尿科評估的重點，不是把每個血精都當成嚴重疾病，而是把少數需要治療或進一步檢查的狀況挑出來。

## 常見原因有哪些？

血精症的原因很多，從很輕微到需要進一步檢查都有。

| 常見原因 | 可能線索 |
| ----- | ----- |
| 攝護腺炎、精囊炎 | 會陰悶痛、射精痛、頻尿、急尿、解尿不適 |
| 尿道炎或性傳染病 | 尿道分泌物、解尿刺痛、近期高風險性行為 |
| 近期醫療處置 | 攝護腺切片、膀胱鏡、導尿、泌尿道手術後 |
| 性行為摩擦或外傷 | 近期性行為較激烈、會陰或生殖器受傷 |
| 攝護腺肥大或血管脆弱 | 排尿症狀、偶發出血、攝護腺相關病史 |
| 結石或泌尿道病灶 | 合併血尿、腰痛、尿路症狀 |
| 凝血異常或藥物 | 使用抗凝血藥、抗血小板藥、容易瘀青或流血 |
| 其他需要進一步檢查的病灶 | 反覆血精、PSA 異常、血尿或其他警訊 |

有時候，檢查後也找不到明確原因。這不代表醫師沒看清楚，而是血精症本來就有一部分屬於暫時性、找不到單一病因的狀況。

## 什麼情況需要特別警覺？

如果你只有一次血精，而且沒有其他不舒服，通常可以先安排門診評估，不一定需要半夜衝急診。

但以下情況建議儘快看泌尿科：

* 血精反覆發作或持續超過數週
* 合併血尿，也就是尿液也看得到血
* 合併會陰痛、睪丸痛、射精痛或下腹痛
* 發燒、畏寒，懷疑感染
* 解尿刺痛、尿道分泌物，懷疑尿道炎或性傳染病
* PSA 異常或有相關高風險病史
* 睪丸摸到腫塊或陰囊明顯腫痛
* 正在使用抗凝血藥或抗血小板藥
* 近期做過攝護腺切片後出血量很多或合併血尿、尿不出來

尤其是「血精加血尿」，要比單純血精更認真評估，因為血液來源可能不是精液路徑，而是整個泌尿道。

## 醫師會怎麼問診？

血精症的問診很重要，因為很多線索藏在細節裡。

醫師可能會問：

* 第一次出現還是反覆？
* 顏色是鮮紅、暗紅還是咖啡色？
* 每次射精都有，還是只有一次？
* 尿液有沒有血？
* 是否合併解尿疼痛、頻尿、急尿、會陰痛、射精痛？
* 最近有沒有新的性伴侶或無保護性行為？
* 最近有沒有攝護腺切片、膀胱鏡、導尿或泌尿手術？
* 有沒有外傷、劇烈運動、性行為摩擦較強？
* 有沒有使用抗凝血藥、抗血小板藥？
* 是否有 PSA 異常或相關高風險病史？

另外，也要先確認是不是「假性血精」。

有時候病人以為是精液有血，其實血是來自尿道、血尿，或來自伴侶的陰道出血。這些情況的檢查方向會不一樣。

## 需要做哪些檢查？

檢查會依症狀、是否反覆、是否合併血尿、感染風險、用藥與過去病史決定，不是每個人都要做一整套。

常見檢查包括：

| 檢查 | 目的 |
| ----- | ----- |
| 尿液檢查 | 看是否有血尿、白血球、感染或結晶 |
| 尿液培養 | 懷疑泌尿道感染時找病菌 |
| 性傳染病篩檢 | 有尿道分泌物、解尿刺痛或風險性行為時評估 |
| 攝護腺指診 | 評估攝護腺大小、壓痛、硬塊或可疑變化 |
| PSA | 反覆血精、攝護腺檢查異常或有相關風險時評估 |
| 陰囊檢查或超音波 | 合併睪丸痛、腫塊或陰囊異常時 |
| 經直腸超音波或骨盆影像 | 反覆或持續血精，評估攝護腺、精囊或射精管 |
| 膀胱鏡 | 合併血尿或懷疑尿道、膀胱病灶時 |

如果你是剛做完攝護腺切片後出現血精，這常見且可能持續一段時間；但若出血量很大、合併發燒、血尿很多、尿不出來，仍需要回診或急診評估。

## 血精症怎麼治療？

治療方式取決於原因。

### 1. 偶發、無症狀：觀察與安心

只出現一次、沒有疼痛、沒有血尿、尿液檢查也沒有異常時，常見做法是衛教、觀察與追蹤。

有時候身體自己修復，之後幾次射精顏色慢慢變淡，就不再出現。

### 2. 發炎或感染：針對病因治療

如果懷疑攝護腺炎、精囊炎、尿道炎或性傳染病，醫師會依檢查結果使用抗生素或其他藥物。若是性傳染病，伴侶是否需要一起檢查與治療，也要一起討論。

### 3. 合併疼痛或排尿症狀：處理攝護腺與骨盆腔問題

有些血精和慢性攝護腺炎、慢性骨盆腔疼痛、骨盆底肌緊繃有關，治療可能包含消炎止痛、改善排尿症狀、生活調整或骨盆底放鬆訓練。

### 4. 醫療處置後：依術後情況追蹤

攝護腺切片、膀胱鏡、導尿或手術後出現血精，常和局部組織刺激有關。多數會逐漸改善，但仍要依醫師告知的術後警訊回診。

### 5. 發現可疑病灶：進一步處理

如果檢查發現 PSA 異常、攝護腺硬塊、睪丸腫塊、泌尿道腫瘤或其他病灶，就會依照原因安排進一步影像、切片或專科治療。

## 生活上可以注意什麼？

血精發生後，最重要的是不要恐慌，也不要完全忽略。

可以先做幾件事：

* 記錄出現次數、顏色與是否每次射精都有
* 觀察尿液是否也有血
* 留意是否有發燒、會陰痛、睪丸痛、解尿刺痛
* 暫時避免過度激烈或造成疼痛的性行為
* 有高風險性行為時，避免無保護性行為並安排篩檢
* 不要自行停用抗凝血藥，需和開藥醫師討論

如果你很焦慮，直接看泌尿科並不丟臉。血精症最常見的問題之一，就是病人被自己腦中的最壞劇本嚇了好幾天。

## 守護者的叮嚀

精液有血，畫面確實很衝擊。

但從泌尿科角度來看，血精症多數時候不是世界末日。真正需要做的，是分辨它是一次性的良性出血，還是身體在提醒你：攝護腺、儲精囊、尿道、睪丸或泌尿道需要被檢查。

找泌尿科醫師把原因釐清，通常比在搜尋結果裡越查越焦慮來得有效。
