> **摘要：** 攝護腺指診，俗稱肛門指診，是泌尿科醫師非常重要的基本功。即使 PSA、超音波、電腦斷層與 MRI 愈來愈進步，有些臨床線索仍然需要靠醫師的手去感覺。這項檢查不只是摸攝護腺大小，也能評估硬塊、壓痛、肛門括約肌力量，以及肛門直腸周邊是否有異常。
> 本文由泌尿科專科醫師周孟翰，用臨床故事和大家聊聊：為什麼今天仍然需要攝護腺指診。

## 泌尿科醫師的黃金手指

攝護腺指診，俗稱肛門指診。

在 MRI、電腦斷層、超音波愈來愈進步的今天，這項看似簡單的檢查，仍然是泌尿科醫師非常重要的基本功。

泌尿科醫師的手指，摸到的不只是攝護腺。有時候，也會摸到病人不好意思說出口的狀況。

## 一個住院醫師時期的難忘經驗

還記得自己剛當住院醫師那年，曾經在肛門指診裡摸到一個奇怪的東西。

圓柱狀。  
中空。  
邊緣相當規則。

老實說，當下我根本不知道那是什麼。

直到後來和大腸直腸外科醫師一起取出後才發現，那是一個鼻噴劑的蓋子。

從住院醫師時期開始，我一直保留著一個習慣：只要遇到男性病人因排尿症狀就醫，不論是在病房會診還是門診，我都會替病人進行肛門指診。

多年後，這個看似再平凡不過的習慣，也曾經幫助我提早發現一位攝護腺癌病人。

## 那位 50 歲出頭、背痛好幾個月的病人

幾年後，我在花蓮急診室遇到一位 50 歲出頭的男性。

他沒有跌倒。  
沒有外傷。  
卻持續被數個月以來的背痛困擾著。

當時不知道為什麼，腦中忽然閃過一個念頭：會不會是攝護腺癌？

於是我替他做了肛門指診。

手指碰到攝護腺的瞬間，心裡其實已經有了答案。

那不是正常攝護腺該有的觸感。摸起來像鵝卵石一般堅硬，失去了原本應有的彈性。

後續抽血、影像檢查與切片結果，最終證實了我的擔心。病人確診為攝護腺癌，並開始接受完整治療。

這不是要說「每個背痛都是攝護腺癌」，也不是說指診可以取代 PSA、MRI 或切片。而是臨床上有些第一個線索，真的可能來自醫師指尖傳來的那一點點感覺。

## 為什麼有 PSA、超音波、MRI，還需要指診？

很多人會覺得：

「現在不是有 PSA、超音波、核磁共振，為什麼還需要肛門指診？」

原因其實很簡單。

有些資訊，機器看得到。  
但有些資訊，只有醫師的手摸得到。

透過指診，除了能評估攝護腺大小，還能感受：

* **攝護腺是否有異常硬塊或可疑變化**：例如局部變硬、不對稱、表面不規則
* **肛門與直腸周圍是否有異常狀況**：例如腫塊、息肉樣病灶或其他摸得到的異常
* **肛門括約肌力量是否正常**：可作為神經功能評估的一部分
* **攝護腺是否出現壓痛**：可能是攝護腺發炎或骨盆腔疼痛評估的重要線索

其中攝護腺發炎，更是成年男性都可能遇到的問題。病人可能有會陰悶痛、解尿不適、頻尿、下腹不舒服，或射精後不適；這時候指診摸到的壓痛與緊繃感，常能提供很有幫助的臨床資訊。

## 攝護腺指診怎麼做？

攝護腺位在膀胱下方、直腸前方，所以醫師可以透過肛門與直腸，摸到攝護腺的後方表面。

檢查時通常會：

1. 請病人採取側躺、膝蓋微彎，或依診間情況調整姿勢。
2. 醫師戴手套並使用潤滑劑。
3. 以手指進入肛門，評估肛門括約肌張力、直腸周邊狀況與攝護腺觸感。
4. 檢查時間通常很短，多數人是壓迫感或異物感，不一定會疼痛。

如果你有痔瘡急性發作、肛裂、肛門疼痛、近期肛門手術，或對檢查非常緊張，請先告訴醫師。這不是考驗忍耐力，檢查本來就應該在尊重與溝通下進行。

## 哪些情況，醫師可能會建議指診？

常見情況包括：

| 情境 | 指診可能提供的資訊 |
| ----- | ----- |
| 排尿困難、尿流變細、夜尿 | 評估攝護腺大小、是否明顯肥大 |
| PSA 異常或攝護腺癌風險評估 | 觸摸是否有硬塊、不對稱或可疑區域 |
| 會陰痛、下腹痛、射精痛 | 評估攝護腺壓痛，輔助判斷攝護腺發炎或骨盆腔疼痛 |
| 懷疑神經功能問題 | 觀察肛門括約肌力量與反射線索 |
| 肛門直腸周邊症狀 | 初步評估是否有可觸摸的腫塊或異常 |

指診不是每個人每次看診都一定要做，但當症狀、年齡、病史或 PSA 結果提示需要進一步評估時，它仍然是很有價值的一步。

## 守護者的叮嚀

醫學一直在進步，檢查儀器也愈來愈精密。

但直到今天，攝護腺指診仍然是泌尿科訓練中不可或缺的基本功。

因為有時候，診斷疾病的第一個線索，不是來自螢幕上的影像，而是來自醫師指尖傳來的那一點點感覺。
