> **摘要：** 50 歲後，男性面臨攝護腺肥大（良性前列腺增生）、攝護腺癌風險增加、勃起功能障礙、夜尿加重等多重健康挑戰。
> 許多患者延遲就醫，誤以為這些是「正常衰老」，導致可預防的併發症（如尿滯留、腎功能惡化）發生。
> 本文由泌尿科專科醫師周孟翰說明 50 歲後每年應進行的泌尿道與性功能篩檢、攝護腺癌早期發現的重要性，
> 以及新店地區熟齡男性如何建立個人化的年度追蹤計畫。

## 「50 歲以後，越來越常起床夜尿......」

「最近夜尿變成每晚 2–3 次，白天上班時頻尿也變嚴重，必須常請假去廁所。」\
「朋友說他爸爸也是這樣，說是『年紀到了』，但我還是有點擔心......」\
「還有性方面，勃起不如以前硬，最近也不太感興趣。這是正常的嗎？」

在新店、大坪林、七張一帶的診所門診中，45–55 歲男性的這些提問非常常見。大多數患者認為這些是「正常老化」的表現，但事實是，**許多症狀是可以改善甚至預防的**，關鍵在於早期發現。

## 50 歲後男性的常見泌尿與性健康問題

### 1. 良性攝護腺肥大（BPH, Benign Prostatic Hyperplasia）

攝護腺會隨著年紀增大。50 歲男性中 50% 有組織學上的肥大，80 歲時更達 80%。

**症狀包括：**

* **下尿路症狀（LUTS）**：
  * 頻尿（≥ 8 次/日）
  * **夜尿**（≥ 2 次/夜）——嚴重影響睡眠品質
  * 尿急感、尿流變弱、排尿不乾淨感
  * 尿後滴漏
* **嚴重時**：急性尿滯留（無法排尿）需導尿

### 2. 攝護腺癌

台灣男性攝護腺癌發病率逐年上升，**50 歲以上是高危險年齡**。

**危險因素：**

* 年齡 ≥ 50 歲
* 家族史（父親或兄弟患攝護腺癌，風險升高 2–3 倍）
* 種族（非洲裔風險最高，其次亞洲人）

**早期攝護腺癌多無症狀**——這是為什麼年度篩檢如此重要。

### 3. 勃起功能障礙（ED, Erectile Dysfunction）

50 歲男性中約 50% 有輕度 ED，60–70 歲更達 70%。ED 常是心血管疾病的早期警示信號。

### 4. 低睪固酮症（已在前文詳述）

部分患者合併睪固酮低下，導致性慾減低、疲倦加重。

## 50 歲後每年應進行的健康篩檢

### 第一步：病史評估與症狀篩查

| 篩檢項目 | 評估內容 |
|---|---|
| **泌尿道症狀** | IPSS 評分（國際攝護腺症狀評分）；夜尿頻率；尿流力 |
| **性功能** | IIEF 評分（國際勃起功能評分）；射精是否困難 |
| **排尿習慣** | 排尿是否費力；是否有尿滯留感 |
| **家族史** | 是否有攝護腺癌、大腸癌、心臟病家族史 |
| **心血管風險** | 血壓、血糖、血脂控制狀況 |

### 第二步：臨床檢查

* **肛門指診（DRE）**：檢查攝護腺大小、硬度、是否有可疑結節
* **腹部超音波或經直腸超音波**：測量攝護腺體積、評估膀胱餘尿量
* **尿流速檢查**：測量排尿力道，評估尿路梗阻程度

### 第三步：實驗室檢查

| 檢查項目 | 頻率 | 用途 |
|---|---|---|
| **PSA（攝護腺特異抗原）** | 每年 | 攝護腺癌篩檢；與年齡、家族史結合評估 |
| **遊離 PSA 比率** | 若 PSA 4–10 ng/mL | 區別良性肥大與癌症風險 |
| **血清睪固酮** | 每 2–3 年，或若有 ED/疲倦時 | 評估是否需 TRT |
| **全血球計數（CBC）** | 每年 | 評估貧血、感染風險 |
| **腎功能（肌酐酸、eGFR）** | 每年 | 長期尿滯留可能傷害腎臟 |
| **血脂與血糖** | 每年 | 評估心血管風險 |

### PSA 篩檢的年齡建議（2025 年 AUA / EAU 指引）

| 年齡與風險 | 建議 |
|---|---|
| **50–70 歲，平均風險** | 每 2 年進行 PSA 檢查；討論篩檢利弊 |
| **40–50 歲，高風險**（家族史、非洲裔） | 每 1–2 年進行 PSA 檢查 |
| **70 歲以上** | 視個人健康狀況與預期壽命；一般不建議常規篩檢 |
| **PSA < 1.0 ng/mL** | 低風險患者可延長篩檢間隔至 2–3 年 |

## PSA 值該如何解讀？

**正常值**：< 4.0 ng/mL

**但 PSA 不是完美的指標**——PSA 升高可能源於：

* 攝護腺肥大（良性）
* 攝護腺炎（感染）
* 攝護腺癌（惡性）——需進一步評估

**若 PSA 升高，醫師會**：

1. 檢查 PSA 的上升速度（PSA Velocity）
2. 計算遊離 PSA 比率
3. 進行肛門指診確認
4. 若仍有疑慮，安排攝護腺磁振造影（MRI）或超音波引導切片

## 攝護腺肥大（BPH）的治療選擇

### 輕度症狀（不影響生活）

* 定期追蹤，暫不用藥
* 調整生活習慣：限制夜間飲水、避免利尿飲料

### 中度症狀（影響日常生活）

| 治療方式 | 說明 |
|---|---|
| **Alpha 阻斷劑**（如 Doxazosin、Terazosin） | 放鬆攝護腺與膀胱頸肌肉，改善尿流；見效快（數天內） |
| **5-Alpha 還原酶抑制劑**（如 Finasteride） | 長期縮小攝護腺；需數月才見效，但效果持久 |
| **複合療法** | 同時使用兩種藥物，效果更佳 |

### 重度症狀或藥物無效

* **經尿道攝護腺切除術（TURP）**：內視鏡刮除肥大組織
* **雷射手術**：綠光雷射或 Holmium 雷射汽化攝護腺
* **微創治療**：如尿道支架置入（針對無法手術患者）

## 性功能的年度評估

50 歲後性功能變化是常見但可處理的問題。

**評估重點：**

* 勃起硬度是否下降
* 射精是否困難或延遲
* 性慾是否明顯減低
* 是否對性行為感到焦慮或挫折

**治療選項包括：**

* PDE5 抑制劑（威而鋼、犀利士等）：有效率 70–80%
* 睪固酮補充（若確認低下）
* 心理治療或伴侶溝通
* 生活方式改善（運動、控制血壓與血糖）

> ED 常是心血管疾病的早期警示。若出現 ED，應同時評估心臟健康。

## 新店地區 50+ 男性的年度追蹤計畫範例

**50–60 歲，無症狀、無家族史：**

* 初次：完整 DRE、腹部超音波、PSA、血液檢查
* 之後每 2 年：PSA、血液檢查、症狀評估

**60–70 歲，或有症狀：**

* 每年：PSA、IPSS 評分、腎功能、性功能評估
* 必要時：經直腸超音波、尿流速檢查

**70 歲以上，且預期壽命 > 10 年：**

* 根據前期 PSA 值與症狀，決定是否繼續篩檢
* 重點轉向症狀改善與生活品質

## 什麼情況應該就醫？

| 情況 | 建議 |
|---|---|
| 50 歲以上，未曾進行攝護腺篩檢 | 泌尿科門診，進行初次完整評估 |
| 夜尿加重（≥ 2 次/夜）、頻尿影響工作 | 評估攝護腺肥大與治療選項 |
| PSA 升高或上升趨勢明顯 | 進一步評估，排除攝護腺癌 |
| 勃起功能明顯下降 | 評估是否需藥物治療；同時檢查心血管健康 |
| 家族中有攝護腺癌病史 | 提早開始篩檢（40–45 歲），頻率更密集 |
| 攝護腺肥大藥物無效或難以忍受副作用 | 討論手術治療選項 |

## 周醫師的提醒

50 歲並不是「人生的下坡路」，而是更需要積極自我照顧的轉折點。許多新店、大坪林、七張地區的患者因為掉以輕心，導致原本可以輕鬆改善的問題（如夜尿、勃起功能）逐漸惡化，甚至引發嚴重併發症。

年度篩檢不是浪費時間，而是提早發現問題、預防後期併發症的最佳投資。攝護腺肥大、性功能障礙、甚至早期攝護腺癌，在診斷初期通常都有多元的治療選擇。延遲診斷，選擇就會越來越少。

如果你在新店、大坪林、七張、土城一帶，今年滿 50 歲或尚未進行過完整的泌尿道篩檢，歡迎到新店高美泌尿科診所預約。周孟翰醫師會為你進行個人化的年度健康評估，根據你的風險因素與症狀制訂最適合的追蹤計畫，讓你在熟齡階段仍能維持良好的生活品質與性功能。
