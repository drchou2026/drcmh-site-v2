> **摘要：** 夢遺 nocturnal emission，是睡眠中自然發生的射精或性高潮反應。它常見於青春期與年輕男性，但成年後也可能偶爾發生。多數夢遺不是疾病，也不代表腎虛、精液流失過多或性功能變差。真正需要注意的是：有沒有血精、疼痛、尿道分泌物、解尿刺痛、發燒，或因頻繁夢遺造成明顯焦慮與睡眠困擾。
> 本文由泌尿科專科醫師周孟翰說明夢遺的常見原因、和晨勃的差異，以及什麼情況需要就醫。

## 診間故事：醫師，我是不是太頻繁了？

「醫師，我有一件事有點不好意思問。」

「沒關係，你慢慢說。」

「我最近睡覺會夢遺。有時候一個月一兩次，有時候壓力大、睡不好，好像又更容易發生。這樣是不是身體太虛？還是性功能有問題？」

有些人是青春期男生，由父母陪來，表情尷尬到想鑽進診間地板。\
也有些人已經成年，甚至有固定伴侶，突然又遇到夢遺，會擔心：「以前不太會這樣，現在突然發生，是不是身體哪裡出問題？」

我通常會先說：

「如果只是睡眠中偶爾射精，沒有疼痛、沒有血、沒有尿道分泌物，大多數時候，這是正常生理現象。」

夢遺不是身體出錯。很多時候，它只是身體在睡眠中完成一次自然的反射。

## 夢遺是什麼？

夢遺，英文常稱 nocturnal emission，也有人稱 wet dream。

簡單說，就是人在睡眠中出現性興奮、性高潮或射精。醒來後可能會發現內褲、睡褲或床單有精液，也可能在過程中醒來。

夢遺可以和春夢有關，但不是每一次都一定記得有夢。很多人醒來只發現「好像發生過什麼事」，但完全想不起夢的內容，這也很常見。

## 為什麼睡覺時會射精？

睡眠不是完全關機。

人在睡覺時，大腦、自律神經、荷爾蒙和生殖器血流仍然會變化。特別是在快速動眼期睡眠，身體可能出現夜間勃起、陰莖血流增加，甚至在某些情況下引發射精反射。

你可以把夢遺想成一個「睡眠中的自動排程」：

* 大腦在做夢或進入特定睡眠階段
* 自律神經活性改變
* 陰莖與生殖器血流增加
* 性刺激記憶、夢境或身體摩擦可能加入影響
* 最後可能出現射精

它不是你醒著時刻意控制的行為，因此不需要過度解讀。

## 夢遺常見在什麼時候？

夢遺最常被注意到的時期，是青春期到年輕成人階段。

原因很直覺：這個階段荷爾蒙變化明顯，性發育正在進行，身體對性刺激與睡眠中生理變化比較敏感。

但夢遺不是青春期限定。成年男性也可能發生，尤其在以下狀況比較容易遇到：

| 狀況 | 可能原因 |
| ----- | ----- |
| 一段時間沒有射精 | 禁慾時間較長，身體自然排出精液的機會增加 |
| 壓力大或睡眠品質不穩 | 睡眠週期與自律神經較容易波動 |
| 最近性刺激增加 | 例如親密互動、影像刺激、性夢內容增加 |
| 睡姿或衣物摩擦 | 外在刺激可能影響生殖器反射 |
| 青春期或年輕成人 | 荷爾蒙活性較高，夢遺較常見 |

頻率差異很大。有人一輩子很少夢遺，有人青春期常發生，也有人成年後偶爾才遇到。只看次數，通常不能直接判斷健康好壞。

## 夢遺和晨勃一樣嗎？

不完全一樣。

晨勃是夜間勃起在醒來時剛好被發現，重點是「勃起」。\
夢遺的重點是睡眠中發生「射精或性高潮反應」。

兩者可能同時出現，也可能各自單獨發生。

| 項目 | 晨勃 | 夢遺 |
| ----- | ----- | ----- |
| 主要表現 | 醒來時陰莖勃起 | 睡眠中射精或性高潮 |
| 是否一定有射精 | 不一定 | 男性通常會有精液排出 |
| 常見關聯 | REM 睡眠、血管神經功能、荷爾蒙節律 | 睡眠階段、性夢、荷爾蒙、禁慾時間、身體刺激 |
| 健康意義 | 長期明顯消失可評估性功能與代謝健康 | 多數偶發夢遺是正常生理現象 |

所以，有晨勃不一定會夢遺；夢遺也不代表晨勃異常。

## 夢遺是不是代表性功能障礙？

通常不是。

夢遺是身體在睡眠中發生的自然反射，和你醒著時的性功能不能直接畫上等號。

它不等於：

* 勃起功能障礙
* 早洩
* 精液太多或太少
* 腎虛
* 攝護腺一定有問題
* 意志力不夠

但如果你同時出現勃起硬度下降、性慾明顯降低、晨勃長期消失、射精疼痛或會陰不適，就可以一起評估性功能、荷爾蒙與攝護腺狀況。

## 夢遺太頻繁怎麼辦？

先不要急著把它當成疾病。

可以先觀察幾件事：

* 最近是不是壓力大、睡眠破碎
* 是否長時間沒有射精
* 是否睡前接觸較多性刺激內容
* 是否穿太緊的內褲或睡姿容易摩擦
* 是否合併疼痛、血精、尿道分泌物或解尿不適

如果只是偶爾夢遺，沒有其他症狀，通常不需要吃藥。\
如果頻率讓你焦慮，可以先從規律睡眠、降低睡前刺激、放鬆壓力、穿著舒適衣物開始調整。

## 什麼情況需要看泌尿科？

夢遺本身多數不需要治療，但以下情況建議就醫：

* 精液出現紅色、粉紅色、咖啡色或血塊
* 射精時疼痛，或夢遺後會陰、睪丸、下腹疼痛
* 合併尿道分泌物、解尿刺痛、頻尿或急尿
* 合併發燒、畏寒或明顯感染症狀
* 近期有高風險性行為，擔心性傳染病
* 夢遺頻繁到嚴重影響睡眠、情緒或日常生活
* 合併勃起硬度下降、性慾低下或晨勃長期消失

這時候要評估的不是「夢遺本身」，而是背後有沒有其他問題，例如攝護腺炎、精囊炎、尿道炎、性傳染病、血精症、睡眠壓力或荷爾蒙狀況。

## 守護者的叮嚀

夢遺不用感到尷尬，也不是身體出了什麼狀況。

對很多男性來說，它只是成長過程或生活某個階段中會出現的自然現象。重點是，不需要因為偶爾夢遺就過度擔心；但如果合併疼痛、血精或解尿異常，就應該進一步評估。

如果只是偶爾發生，沒有疼痛、沒有血、沒有解尿症狀，多數可以放心觀察。\
如果合併血精、疼痛、尿道分泌物、感染風險，或夢遺造成你很大的心理壓力，請讓泌尿科醫師協助你把問題釐清。
