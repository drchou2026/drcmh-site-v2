> **摘要：** 適尿通（dutasteride）是泌尿科常用的攝護腺肥大藥物，屬於 5α 還原酶抑制劑，可降低二氫睪固酮（DHT）。DHT 會刺激攝護腺增生，也會讓具遺傳敏感性的毛囊逐漸迷你化，因此同一套機轉也被延伸到雄性禿治療。本文由泌尿科專科醫師周孟翰說明適尿通在攝護腺肥大與生髮治療中的角色、效果期待、性功能副作用發生率，以及在新店、文山區就醫時該如何和醫師討論。

## 跨界打怪的泌尿科藥物：同一把鑰匙，開兩扇門

「醫師，我朋友說攝護腺肥大的藥也可以長頭髮，是真的嗎？」\
「我本來是想治療掉髮，但看到副作用寫性功能障礙，有點不敢吃。」

這類問題在新店、大坪林、景美、木柵一帶的泌尿科門診越來越常見。原因很簡單：現代男性越來越重視外觀，也更願意面對排尿與性功能問題。適尿通（dutasteride）剛好站在這幾個議題的交叉點。

但第一個觀念要先講清楚：**適尿通的主要治療場景是攝護腺肥大，不是單純的保養品或生髮保健食品。** 在台灣，適尿通用於雄性禿治療並不是藥品仿單明確標示的核准適應症，屬於 **off-label use（仿單標示外使用）**。它仍是處方藥，需要評估適應症、副作用、PSA 追蹤與個人風險。

## 適尿通治療攝護腺肥大的機轉

攝護腺肥大（Benign Prostatic Hyperplasia, BPH）和男性荷爾蒙有密切關係。睪固酮本身不是唯一主角，真正對攝護腺和毛囊影響更強的是 **DHT（二氫睪固酮，Dihydrotestosterone）**。

DHT 是由睪固酮經過 **5α 還原酶（5-alpha reductase）** 轉換而來。人體主要有第 1 型與第 2 型 5α 還原酶：

| 類型        | 常見分布             | 臨床意義              |
| --------- | ---------------- | ----------------- |
| 第 1 型 5α 還原酶 | 皮膚、皮脂腺、毛囊等       | 與頭皮油脂、毛囊環境相關      |
| 第 2 型 5α 還原酶 | 攝護腺、外生殖器、部分毛囊    | 與攝護腺增生、雄性禿高度相關    |

Dutasteride 的特色是**同時抑制第 1 型與第 2 型 5α 還原酶**，因此可大幅降低 DHT。當 DHT 下降後，攝護腺受到的荷爾蒙刺激減少，長期下來可讓攝護腺體積縮小、改善尿道壓迫。

### 對攝護腺肥大的實際效果

適尿通不是「吃了今天晚上尿流就變強」的藥。它的效果通常需要時間累積：

* **3–6 個月**：排尿症狀逐漸改善
* **6–12 個月**：攝護腺體積縮小效果更明顯
* **長期使用**：可降低急性尿滯留與未來手術風險

臨床上，若患者主訴夜尿、尿流變細、排尿費力或餘尿感，泌尿科醫師可能會考慮評估攝護腺大小、尿液檢查、PSA、超音波與症狀嚴重度。對攝護腺體積較大的患者，dutasteride 這類藥物通常更有角色；實際仍需由專業醫師判斷攝護腺與下泌尿道症狀的關聯性。

## 為什麼同一顆藥會被拿來談雄性禿？

雄性禿（Androgenetic Alopecia）不是單純「頭皮缺營養」，而是毛囊對 DHT 特別敏感。具遺傳體質的男性，前額、髮際線與頭頂毛囊在 DHT 影響下會逐漸迷你化：

1. 毛髮生長期變短
2. 頭髮變細、變軟、變短
3. 毛囊逐漸縮小
4. 外觀看起來髮量變少、髮線後退

因此，降低 DHT 就成為治療雄性禿的重要策略。常見藥物包括：

| 藥物                         | 主要作用                 | 常見用途                 |
| -------------------------- | -------------------- | -------------------- |
| **Finasteride**            | 主要抑制第 2 型 5α 還原酶    | 雄性禿、攝護腺肥大（不同劑量）    |
| **Dutasteride（適尿通）**      | 抑制第 1 型與第 2 型 5α 還原酶 | 攝護腺肥大；部分情境用於雄性禿討論 |
| **Minoxidil**              | 促進毛囊生長期、增加局部血流      | 外用生髮常用藥              |

Dutasteride 抑制 DHT 的能力通常比 finasteride 更強，因此在雄性禿研究中，常可看到髮量、髮徑或照片評估改善。不過，**效果較強不等於每個人都該直接使用**。特別是當 dutasteride 用於雄性禿並未被藥品仿單明確列為核准適應症時，臨床上會把它視為 **off-label use（仿單標示外使用）**：代表醫師可能基於研究證據與個別狀況判斷後使用，但必須先和患者充分討論預期效果、替代方案與潛在副作用。年輕男性若只是輕度掉髮，通常仍需先確認診斷、掉髮型態、家族史、頭皮狀況與生育計畫，再決定是否使用口服 5α 還原酶抑制劑。

## 生髮治療時可以期待什麼效果？

若雄性禿診斷明確，dutasteride 的核心目標不是讓所有人「變回高中髮量」，而是：

* 減少 DHT 對毛囊的持續攻擊
* 延緩髮際線與頭頂稀疏惡化
* 讓部分迷你化毛囊恢復較粗的髮徑
* 提高整體視覺髮量

一般會用 **6–12 個月** 來觀察治療反應。前 1–3 個月若覺得掉髮沒有立刻停止，不代表治療失敗；毛囊週期本來就需要時間。

但如果已經是完全光滑、毛囊萎縮很久的區域，單靠藥物通常很難「憑空長回」。這時可能需合併外用 minoxidil、低能量雷射、PRP 或植髮評估。若同時有排尿症狀或性功能疑慮，建議找熟悉男性荷爾蒙與泌尿系統的醫師一起評估，避免只看到頭髮，卻忽略攝護腺與性功能。

## 適尿通可能有哪些副作用？

適尿通多數患者可穩定使用，但它畢竟會影響 DHT，因此需了解可能副作用：

| 副作用類型        | 可能表現                         | 建議處理方式                      |
| ------------- | ---------------------------- | --------------------------- |
| 性慾下降          | 對性行為興趣降低                     | 回診評估時間點、壓力、睡眠、睪固酮與用藥關聯     |
| 勃起功能障礙        | 硬度下降、維持困難                    | 評估血糖血脂、血壓、抽菸、睪固酮；必要時治療 ED |
| 射精改變          | 精液量減少、射精感改變                  | 多數不影響健康，但若備孕需提前討論           |
| 乳房不適或腫脹       | 乳房壓痛、男性女乳                    | 回診檢查，排除其他乳房病灶               |
| PSA 數值下降      | **PSA 約可下降一半，影響攝護腺癌篩檢解讀**     | 抽 PSA 時務必告知正在服用 dutasteride   |
| 懷孕女性接觸風險      | 膠囊破損內容物可能影響男性胎兒外生殖器發育       | 孕婦或可能懷孕女性避免接觸破損膠囊           |

另外，dutasteride 半衰期長，停藥後不會立刻完全代謝。若因副作用需要調整，應和醫師討論，不建議自行忽停忽吃。

## 性功能障礙發生率：到底多常見？

這是大家最在意，也最容易被網路放大的問題。

在攝護腺肥大臨床試驗中，dutasteride 常見性功能相關副作用多集中在用藥初期。以藥品仿單資料來看，前 6 個月大約可見：

| 項目       | Dutasteride 組 | 安慰劑組 |
| -------- | -------------- | ---- |
| 勃起功能障礙   | 約 4–5%         | 約 2% |
| 性慾下降     | 約 3%           | 約 1% |
| 射精障礙     | 約 1–2%         | \<1% |

用於雄性禿族群的研究中，因患者較年輕、基礎性功能較好，數據會因研究設計不同而有差異。有研究顯示，dutasteride 組性功能相關不良事件約 **16%**，安慰劑約 **8%**，多為輕中度且可恢復；但這類研究樣本數相對有限，解讀時不能只看單一數字。

更貼近臨床的說法是：**性功能副作用確實可能發生，但不是每個人都會發生；發生後也不代表只能硬撐。**

## 如果發生性功能障礙，該怎麼辦？

若開始服用適尿通後出現性慾下降、硬度變差或射精量明顯減少，建議不要急著把所有問題都歸咎於藥物，也不要自行購買來路不明的壯陽藥。泌尿科評估會比較完整：

### 1. 先確認時間關係與嚴重度

* 是用藥後幾週內出現，還是本來就有？
* 是性慾下降，還是勃起硬度下降？
* 是偶發，還是連續 3 個月以上？
* 有沒有壓力、失眠、酒精、抽菸或伴侶關係因素？

### 2. 檢查常見共病

勃起功能障礙常是血管健康警訊。建議視情況檢查：

* 血糖與糖化血色素
* 血脂
* 血壓
* 睪固酮
* PSA 與攝護腺狀況

### 3. 調整治療策略

可和醫師討論：

* 觀察一段時間，確認是否逐漸改善
* 調整用藥或停藥評估
* 若主要目標是生髮，改以外用 minoxidil 或其他治療搭配
* 若主要目標是攝護腺肥大，評估 alpha-blocker、每日低劑量 tadalafil、微創手術或雷射手術
* 若確認合併勃起功能障礙，可使用 PDE5 抑制劑治療，例如 sildenafil 或 tadalafil
* 若合併睪固酮低下，需另行評估男性荷爾蒙治療是否適合

### 4. 有備孕需求要先講

Dutasteride 可能造成精液量下降，也可能影響部分男性對性功能的主觀感受。若近期有備孕計畫，建議在用藥前就和醫師討論。對年輕男性而言，「頭髮、性功能、生育計畫」常常要一起放進治療決策，而不是分開看。

## 新店、文山區男性就醫時，該怎麼和泌尿科醫師討論？

如果你住在新店、大坪林、七張、安坑，或文山區的景美、木柵、萬芳一帶，想討論適尿通、攝護腺肥大、雄性禿與性功能副作用，門診時通常會從以下幾個方向一起了解：

| 想討論的問題      | 建議評估的內容                       |
| ------------ | ------------------------------ |
| 排尿症狀         | 夜尿次數、尿流變細、急尿、頻尿、是否曾尿不出來      |
| 掉髮狀況         | 髮際線後退或頭頂稀疏、家族史、已用過哪些生髮產品     |
| 性功能狀況        | 性慾、硬度、射精量是否改變，持續多久            |
| 健檢資料         | PSA、血糖、血脂、肝腎功能、睪固酮            |
| 人生計畫         | 是否備孕、是否重視射精量、是否可接受長期服藥        |

泌尿科的價值不只是開藥，而是把排尿、攝護腺、男性荷爾蒙、性功能與生活品質一起評估。尤其大台北地區許多患者平日工作忙、久坐、壓力大，頻尿、夜尿、掉髮與勃起困難可能同時出現，更需要個人化判斷。

## 周醫師的提醒

適尿通是一個很典型的「跨界藥物」：在泌尿科，它幫助攝護腺肥大的患者降低 DHT、縮小攝護腺、減少尿滯留與手術風險；在雄性禿治療中，它也因為抑制 DHT 的能力受到關注。

但跨界不代表可以隨便用。對有攝護腺肥大症狀的男性，它可能是很適合的長期治療；對單純想治療雄性禿的年輕男性，則需要更細緻地討論性功能、生育計畫與替代選項。
